# 📝 To-Do App (Görev Yönetim Uygulaması)

Modern, hızlı ve kullanıcı dostu bir Görev Yönetim (To-Do) web uygulaması. Bu proje, "Minimum Viable Product" (MVP) prensipleriyle tasarlanmış olup, temel görev takibi ihtiyaçlarını karmaşadan uzak, en yalın haliyle karşılamayı hedefler.

## ✨ Özellikler

* **Dinamik Görev Ekleme:** JavaScript DOM manipülasyonu ile sayfa yenilenmeden anlık görev oluşturma.
* **Görev Etkileşimi:** Tamamlanan görevlerin görsel olarak işaretlenmesi (üstü çizili ve soluk görünüm).
* **Görev Silme:** İhtiyaç duyulmayan veya biten görevleri (Event Delegation kullanılarak) bellekten anında kaldırma.
* **Responsive Tasarım:** Tailwind CSS kullanılarak tamamen mobil uyumlu (Mobile-First) hazırlanan modern kullanıcı arayüzü.

## 🛠️ Kullanılan Teknolojiler

* **Arayüz (UI):** HTML5, Tailwind CSS
* **Mantık & Etkileşim:** Vanilla JavaScript 
* **Versiyon Kontrolü:** Git & GitHub 
* **Geliştirme Standartları:** Conventional Commits (Geleneksel Commit Standartları) kullanılarak disiplinli bir branch/PR akışı izlenmiştir.
